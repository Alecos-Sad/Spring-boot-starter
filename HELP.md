# Написание starter

### В проекте будет два модуля. Один под стартер, второй под сервис. Сделано так для демострации.


Описание шагов.

* Создаем контроллер подписи (signature)
* Создаем сервис
* Запуск сервиса из командной строки: 
 - mvn clean install
 - mvn -pl sample-service spring-boot:run
